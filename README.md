# NORDEX-SHIFTS-ANALYTICS-AND-OPTIMISATION
05:41A cross-functional manufacturing analytics project built with Power BI, DAX and SQLite3. Analysed 13,650 production events across 3 shifts, DAX Measures for calculated OEE and 15+ KPIs benchmarked against Industry 4.0 standards, and delivered 4 interactive dashboards revealing critical performance gaps and actionable insights.
